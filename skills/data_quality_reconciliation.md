# Skill: Data Quality Reconciliation

Use this for support-note verification, attribute checks, and mismatch detection.

Workflow:
1. Treat the note as a claim, not as truth.
2. Find the canonical catalog or commerce record.
3. Compare each claimed attribute against the canonical record.
4. If the exact item is absent, distinguish absence from attribute mismatch.
5. Return the requested yes/no, correction, or explanation with canonical evidence.

Do not infer facts from adjacent similar products unless the canonical record proves them.
