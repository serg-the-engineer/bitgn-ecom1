# Skill: Catalog Lookup And Counting

Use this for catalog presence checks, product identification, and catalog counts.

Workflow:
1. Resolve the exact requested product class, brand, and attributes before answering.
2. Prefer broad discovery first, then narrow verification against authoritative catalog records.
3. For counts, search broadly, read candidate records or reports, deduplicate, then count.
4. If a requested output format is literal, preserve it exactly in the final message.
5. Cite the authoritative records you used; do not cite only a search result.

Do not memorize product names, catalog paths, dates, ids, or prior task answers.
